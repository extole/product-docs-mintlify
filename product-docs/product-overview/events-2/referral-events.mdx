## Referral Program Business Events

[//]: # "What events are part of my referral program flow?"

In a standard Refer a Friend program flow, the customer journey starts with a promotion on your website or in your mobile app. As a customer engages with the promotion and participates in your program, business events help track where they are at in their journey. 

Referral programs typically include the following events:

| Event Name        | What Extole Tracks                                                                                                                          |
| :---------------- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| Promotion Viewed  | The number of impressions on in-app or onsite promotions or emails (does not include links posted on social channels or untracked sources). |
| Promotion Clicked | The number of clicks on in-app promotions, onsite promotions, email promotions, or promotion links posted on social channels                |
| Advocate Viewed   | The number of share experience views.                                                                                                       |
| Viewed            | The number of shares.                                                                                                                       |
| Shared            | The number of views on share emails.                                                                                                        |
| Clicked           | The number of clicks on share links.                                                                                                        |
| Visited           | The number of visits to a share destination.                                                                                                |
| Signed Up         | The number of signups completed.                                                                                                            |
| Friend Rewarded   | The number of rewards issued to friends.                                                                                                    |
| Friend Converted  | The number of conversions made by friends.                                                                                                  |
| Advocate Rewarded | The number of rewards issued to advocates.                                                                                                  |

### Additional Events

Beyond the events listed above, retailers with Refer A Friend programs may also want to track the following additional events:

| Event Name | What Extole Tracks                |
| :--------- | :-------------------------------- |
| Shipped    | The number of purchases shipped.  |
| Returned   | The number of purchases returned. |

## Engagement Program Events

Engagement programs beyond referral, such as Drop A Hint, Friends & Family, and Influencer, will typically use the same core set of business events listed above with slight nuances to some of the event names. For example, in an Influencer program the Advocate Rewarded event is instead called Influencer Rewarded.
