For our customers' marketing programs, Extole provides baseline templates that have been audited for ADA compliance. As part of the implementation process, Extole customizes the templates based on brand-specific fonts, colors, imagery, and so on. It is a best practice for Extole customers to at this point run an ADA-compliance scan, as customizations may introduce areas that require ADA-related updates. The Extole support team can then process any ADA-related feedback, make the appropriate changes, and ensure the program is confidently ready to go live.
