## Overview

[//]: # "How do I upload files to Extole?"

Our Batch Jobs page, located in the My Extole Tech Center, offers a powerful self-service experience for your file upload needs. This page allows you to quickly and reliably upload file-based events and automate jobs of large files without needing to coordinate with our Support team.

## Creating a New Job

[//]: # "How do I create a batch job to upload files to Extole?"

Click on the + New Batch Job button to start the job creation process. A drawer-style window will slide out from the right-hand side of your screen with all of the necessary fields to fill in for your job.

![](https://files.readme.io/0d09cadf5b4deb876637ef4c491db1b437f0c1843e0c7f1eae2e957fd13a08e9-91d4ed5-Screen_Shot_2022-09-19_at_12.23.30_PM.png)

## Searching for a Job

[//]: # "How do I search for an uploaded file in Extole?"

You can now search for your jobs by typing keywords into the search bar and applying search filters. 

![](https://files.readme.io/19f9af33160447fe8e508b69bc53b1b173c7869c82c144651c3b887e3e967200-11d979e-Screen_Shot_2022-09-19_at_12.24.34_PM.png)

Apply filters to your search by changing the “Search by:” dropdown criteria. For example, if you select Search by: User and click on the bar, it will generate a list of users for you to browse. Selecting a user will show you all of the jobs that user has created.

## Sorting Jobs by Status

[//]: # "How do I search for a file based on its upload status?"

In the jobs table, you have the option to sort your jobs based on status. You can select any number of statuses.

![](https://files.readme.io/6e4cd81e3bd7e1717103483e040b0e3fe525ebc7105186c69243bfddedb9b416-1377490-Screen_Shot_2022-09-19_at_12.26.01_PM.png)

### Job Statuses

The job statuses include pending, in progress, done, failed, canceled, and expired.

| Status      | Description                                                                                                                     |
| :---------- | :------------------------------------------------------------------------------------------------------------------------------ |
| Pending     | The job is in our system and getting ready to process. This preparation phase can take up to a couple of minutes.               |
| In Progress | The job is actively being processed by our system. The duration of the processing period is dependent on the size of the file.  |
| Done        | The job has been successfully completed.                                                                                        |
| Failed      | The job was unable to be processed. This is most likely due to issues with your file format or duplicated files.                |
| Canceled    | The job was canceled by you before it was completed.                                                                            |
| Expired     | The job was uploaded more than 90 days ago. These files are removed from our system, but we do retain history that the job ran. |

## Viewing Job Details

[//]: # "How do I details about an uploaded file in Extole?"

Each job in the table has a detailed view. Clicking on a job will open a drawer-type window from the right-hand side of your screen that lists more in-depth information about the job.

![](https://files.readme.io/2a5854cb2eb8985ec5f657b6fb8b1a8bbff33727a5d5e3e20363889eb2deec60-00e353c-Screen_Shot_2022-09-19_at_12.35.27_PM.png)

### Detailed View

Information displayed the detailed view of a job includes the following:

* Name
* Source (clicking this link will download the file)
* Type
* Status
* Valid/Invalid Rows
* Start/Completed Time
* Tags
* Event Names (if all events in your file are the same type)
