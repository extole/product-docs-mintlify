## Can I use a tag manager such as Tealium, Ensighten, or Google Tag Manager?

[//]: # "Can I use a tag manager such as Tealium, Ensighten, or Google Tag Manager?"

Yes. Extole tags can be managed using tag management tools. For zone tags, it sometimes makes more sense to use an ad server instead of a tag manager.

## Does block order matter?

[//]: # "Does block order matter?"

Block order should not matter, as the order of load is very forgiving. Specifically, when Extole's JS library appears, it will continue to scan the entire page for the presence of other tags.

[//]: ___

## Are your tags minified?

[//]: # "Are Extole tags minified?"

Yes. All Extole creative content uses a combination of minification and CDN hosting to make delivery as fast as possible.

[//]: ___

## Is the conversion tag the only way to send Extole information about conversions?

[//]: # "Is the conversion tag the only way to send Extole information about conversions?"

It is also possible to send Extole conversion events through a RESTful API or a file upload to the Extole SFTP server.
