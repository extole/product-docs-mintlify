## Overview

We strive to make the Extole platform as easy to use as possible—but everyone needs a little help sometimes! Whether you're troubleshooting an issue, you have a specific technical request, or you want to customize your program experience, [Extole Help](https://extole.atlassian.net/servicedesk/customer/portal/3) is here to assist you.

## File a Request

To file an inquiry with our support team, start by sending a detailed email to [support@extole.com.](mailto:support@extole.com.) You can also login to myextole.com and utilize our in-app chat for help directly from a Team Member.
