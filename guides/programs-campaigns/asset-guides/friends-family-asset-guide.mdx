# Overview

Give your most important consumers a robust offer with the ability for them to share the exclusive offer with a select number of friends!

An important aspect of the Friends and Family program is the ability to nurture consumers—both advocates and their friends. This program has two primary email to nurture the experience:

- Advocate Earned Reward Email - the original announcement email inviting consumers into the friends and family program with their coupon code

  - These emails are set up to drip to a consumer X number of days after the launch. If you would like these to send on an exact date, please let the Extole team know.

- Friend Earned Reward Email - the "share email" that goes to the invited friend with their coupon code (note: this email replaces the refer a friend "Share Email")

| Name | Recommended Image Size   (width x height) | Location |

| Company Icon | 96 x 96 px | Overview |
| Email Banner Image | 1200 x 700 px | Overview - all emails |
| Microsite Banner Image for Desktop | 2800 x 800 px | Advocate - Microsite |
| Microsite Banner Image for Mobile | 1080 x 900 px | Advocate - Microsite |
| Share Experience Vertical Image for Desktop | 790 x 1300 px | Advocate - Share Experience |
