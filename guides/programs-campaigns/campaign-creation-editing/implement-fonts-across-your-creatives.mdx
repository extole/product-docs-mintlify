# How to Change Fonts within Campaign Editor

You have the ability to implement fonts for your campaigns within the Campaign Editor of your My Extole account. This allows you to match your referral program fonts to the branding on your site.

# Supported Font Types

**Web-Safe Fonts**

These are fonts that are safe to use across all email clients, web browsers, and devices. The most common examples are Arial and Helvetica for sans-serif fonts and Georgia, Times New Roman for serif fonts. You can find a list at: [CSS Font Stack](https://www.cssfontstack.com/).

**Google Fonts**

[Google Fonts](https://fonts.google.com/) are very common and easy to implement. While they are still not 100% supported in all email clients, they offer more support than Custom Web Fonts do. See the chart below to see the supported browsers:

# How to Add your Font to a Campaign

## Overview Section

The Overview section of Campaign Edit is where you will configure the font-family, which will cascade through all creatives. It consists of body font-family and header font-family. The body font will always have a font-weight of 400 / normal. The header font will always have a font-weight of 700 / bold.

## Google Fonts

Follow the instructions under the "How to Select your Google Font" section above, then continue on to do the following to integrate in Campaign Edit:

Copy the URL reference from Google Fonts to use in Campaign Edit—you should have this available if you completed all of the steps successfully in the above section.

In Campaign Edit, paste this URL reference under "Body Google Font Family URL" or "Header Google Font Family URL".

![Screenshot 2024-10-11 at 10.48.11.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421735255/b3e7faaeb96cf8be52a95b539337/34304765426579?expires=1778306400&signature=642ab952f8cf689a84a06284c022eb44b1b5f8b7edc35132cffb9d9a7c5d83f5&req=dSQlF859mINaXPMW3nq%2BgaPVDGW16mb%2BYOopThTC9pkQbOW8i6nDyudJaBfi%0AWqMumipsOGRp9DfphBjthI6gvu0%3D%0A)

Now copy the font family name from Google Fonts.

### Add a Backup Font

Add the font family name in Campaign Edit, but make sure to also add a fallback web-safe font family to ensure that there is a second option in case your Google Font is not supported.

![Screenshot 2024-10-11 at 10.49.13.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421735266/d16440b449dc71af311d668d1352/34304781434771?expires=1778306400&signature=973bc0b4f8eabf27623d696140fb6ff05e954d691f005f5e49130ddb644ed86e&req=dSQlF859mINZX%2FMW3nq%2Bgezk9XDlSIZ89z9b392aSD6ZRrvaZB0XaLQR9KJr%0AAB5P1f%2FjrP5IOJddzrBav2MulWk%3D%0A)

## Web-Safe Fonts

If you are not using Google Fonts and only want to use web-safe fonts, it's very simple.

You can ignore the Google Font URL settings and go straight to the "Body Font Family" / "Header Font Family" settings in the Overview section in Campaign Edit. Add the web-safe font family name in Campaign Edit, and make sure to add an additional fallback font-family. **![Screenshot 2024-10-11 at 10.49.13.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421735256/6c57f3189d8d49bf05a224359565/34304781434771?expires=1778306400&signature=d81e8b659682b1cf1fdb4bf16df6c398f73a7f66f8c0403dcba3a3d9e92b4f43&req=dSQlF859mINaX%2FMW3nq%2Bge%2B1okwfLUn3bAmxrQYP%2FIy%2FnkfomTvxmBjAas2v%0AR3XB5OHPlssKoS5dkQZ2IUzhMTg%3D%0A)** *NOTE: If the font-family has more than one word, you'll need to put it in quotes (ex: 'Times New Roman').*
