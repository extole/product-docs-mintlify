# Overview

The Nomination Report provides valuable insights into the nominations received in your program. By specifying the desired fields and time range, you can generate a comprehensive report that reveals essential details about the nominees, nominators, and associated campaign information. This article will guide you through the process of running the report and interpreting the resulting data.

# How This Report Can Help You

1. **Analyze Nomination Messages**: The report provides valuable insights for evaluating nominations and selecting deserving winners. By analyzing the nomination messages submitted by nominators, you can gain a deeper understanding of the specific reasons and accomplishments that led to the nominations. This information allows you to identify exceptional achievements and customize rewards or recognition accordingly.
2. **Identify Recognition Patterns**: By reviewing the nominations, you can identify patterns and trends in the types of recognition given within your program. This insight can help you understand which individuals or achievements are being recognized most frequently, enabling you to tailor your program to better meet the needs and preferences of your participants.
3. **Recognize Top Nominators**: The report provides details about the nominators, including their names and contact information. By recognizing and acknowledging the most active nominators, you can foster a culture of appreciation and encourage continued participation in your programs.
4. **Understand Nominee Profiles**: The report reveals information about the nominees, including their names, email addresses, and optionally, their occupations or other relevant information, depending on how your nomination campaign or program is configured. With this nominee data at your disposal, you can enrich your CRM profiles by adding these details. This enables you to personalize your interactions and offers with the nominees, creating a more tailored and engaging experience for them.

# How to Run the Report

1. Specify the Time Range: Choose the desired time range for the report, such as the last day, last week, or custom-defined intervals. The report's data is available starting from September 30, 2020
2. Specify the Event Names
3. Customize the report by selecting specific program labels or campaign IDs, or choose "All" to include all programs and campaigns.
4. Advanced Options: In this section, you can choose additional preferences for the report format and integration.

  - Select the desired format (XLSX, JSON, CSV, PSV, CSV - no header row, or PSV - no header row) to export the report in the desired format.
  - You can also integrate the report output with an SFTP server.
  - Mappings: The mappings define the fields and attributes included in the report. These mappings establish connections between the report data and specific identifiers, such as event IDs, event names, event times, quality ratings, nominator and nominee information, and more. The provided list of mappings is comprehensive and covers various aspects of the nomination process.
  - Filters: Optionally, you can apply filters to refine the set of events analyzed in the report. Filters enable you to specify visit types, quality ratings, sources, channels, country, or language, allowing for a more precise analysis of the nominations.
  - Save & Run: After configuring the report parameters, click "Save & Run" to initiate the report generation process.

# Report Output

Once the report is executed, it will generate a table containing the following data columns:

- Event ID: Extole’s Unique identifier for each event in the report.
- Event Name: Name of the event.
- Event Time: Timestamp indicating when the nomination event occurred.
- Quality: The quality score of the nomination event (e.g., high, low).
- Nominator Person ID: Extole's unique ID of the person who made the nomination.
- Nominator Partner User ID: Client's unique ID associated with the nominator.
- Nominator First Name: First name of the nominator.
- Nominator Last Name: Last name of the nominator.
- Nominator Email: Email address of the nominator.
- Nominee First Name: First name of the nominee.
- Nominee Last Name: Last name of the nominee.
- Nominee Email: Email address of the nominee.
- Nominee Occupation: Occupation or professional role of the nominee, if applicable to your program.
- Nomination Message: Message or description provided by the nominator when submitting the nomination.

The report data will contain multiple rows, each representing a separate nomination event. Reviewing this data will provide insights into the nominated individuals, the nominators, and the context surrounding the nominations.
