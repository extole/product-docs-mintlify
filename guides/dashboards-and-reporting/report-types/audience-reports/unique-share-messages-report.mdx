# Overview

See what your advocates are saying when they customize the share message in the widget. While we like showing everything, we will not report messages that were blocked by our quality engine.

# Report Output

This report includes the following data, if available:

- Subject: The subject line of the share. This may not always be present.
- Message: This is the message that an advocate shares with their friend.
- Channel: This is how the advocate shared (email, Facebook, Twitter, etc.).
- Unique Send Count: This is the count of how many times that exact message and subject combination has been shared.
- Sender ID, Sender Partner User ID, Sender Email, Sender First Name, Sender Last Name: This is the person ID (within Extole), partner user ID (usually the user ID in your system), email address, first name, and last name of the advocate who most recently shared that message and subject combination.

  - **Note:** You'll notice that email address is obfuscated. When you run the report from within My Extole, you have the option to show the entire email address, or to obfuscate the first part of the email address and show only the domain. This can be configured in the Advanced section.

- Last Send Date: This is the date of the most recent send of that message and subject combination.
