# Overview

Want to discover insights like what percentage of your customers are sharing on mobile? Dig deeper into your Extole dashboard data with our Metrics report. The Metrics report enables you to break down your program performance by a variety of dimensions while pulling in the counts or rates of the events in your program.

# Use Cases

The following use cases are great examples of when you may want to run a Metrics report.

- Get metrics specific to promotion source and device type.
- Receive weekly reports on the performance of your referral program.
- Discover top promotion sources based on the number of shares.
- Compare performance between your burst and evergreen campaigns.

# How to Run a Metrics Report

Complete the following steps to set up a metrics report.

1. Within your My Extole account, navigate to the Reports tab.
2. Select the + New Report button.
3. Search for the Metrics report and click on it to begin the report creation process.
4. Under **Report Configuration**, name your report and determine if you'd like to run the report once or schedule it so that you receive the data daily, weekly, monthly, etc.![Screen_Shot_2023-03-28_at_9.42.42_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733723/ca8a783bfcab4f03b39b087c7ac8/15263598807315?expires=1778306400&signature=2b9038a555e97975c688c8fede02369ba0a6a28efa7449f86eb9d34021e90526&req=dSQlF859noZdWvMW3nq%2BgQhuaW5bfbeO0RF2g0%2BZQ3zSBGUntouMo7TFBT%2BM%0AWA2b%2FXl8wJEp%2BcPPeZHQbjIaWVc%3D%0A)
5. In the **Filter** section, select the timeframe and specific programs or campaigns for which you'd like to pull data.![Screen_Shot_2023-03-28_at_9.43.38_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733718/3408e4aba70d32d1d8b078b2cc6e/15263654300563?expires=1778306400&signature=3bcd06d4572b732a6b0d945f3e5b67121bd5fd11c11f76234aa46f06e02e9c51&req=dSQlF859noZeUfMW3nq%2BgVrb1xMHh4bzThCOxnrjLReUdhKRbcnbhHKzukDt%0AmnxuloNet3ACuxZBXu7mmjsdDmQ%3D%0A)
6. In the **Report Columns** section, select **+ New Dimension** to organize your data in a variety of ways. Each dimension becomes a column in your report and the values of the dimensions become rows. Remember to be intentional—the more dimensions you add, the more columns will end up in your report.![Screen_Shot_2023-03-28_at_9.44.53_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733722/4b4b79a8940e328a5f10f1ed64e2/15263665184915?expires=1778306400&signature=f3b600cc14bf7b833ff368c56e6fb351cec6cf433658d9bf53fc7b4462ce3c6d&req=dSQlF859noZdW%2FMW3nq%2BgcNyyEBFgfM7TmNHfmtJXGep1pK3N%2Bs9GYiIzC6x%0AmQ94ZZqy19TN%2BHglUjDLybbtiqs%3D%0A)

  - **Example**: If you want a report that breaks down performance by Promotion Source and Device Type, you may want to consider the following dimensions.

    - Period: Would you like to separate out the data by day, week, month, quarter, or year?
    - Campaigns/Programs: Would you like to see key performance data across multiple programs or campaigns in one report?
    - Channel: How are advocates sharing the program with their friends? Do they favor SMS, email, or Facebook?
    - Language and Country: If you're running an international program, do you want to break out programs and campaigns by locals?
    - Device Type: How does each metric compare based on device type? Are more customers engaging with your program on desktop or mobile?

7. Select **+ New** **Step** or **+ New Rate** to add columns that display counts, values, rates, or percentages for each dimension you’ve chosen.

  - Step Options

    - Count of Events: Metrics that aggregate the business events in your program or campaign.
    - Count of People: Metrics that aggregate the number of people that participated in a specific business event.
    - Total Value: Metrics on the revenue or sum of dollars associated with friend purchases, or advocate and/or friend-redeemed rewards.

  - Rate Options

    - Predefined Rate: Key rates that align with the journey of the selected program(s).
    - Custom Rate: Create a custom rate not already predefined in your program’s journey.![Screen_Shot_2023-03-28_at_9.50.26_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733721/c19bdf70fe84a9d26d7f01971a8f/15263868034323?expires=1778306400&signature=2febd67a1d41c9da57bfd4a1326b410dcb1f1e4d631dcb8276a38227d590109a&req=dSQlF859noZdWPMW3nq%2BgVY8g%2BKS33NokRUXwIC0MuvxIMwqYDXj7e9aHWF0%0AtLEwLe1P3LTfkflK2zDXYY3NaMc%3D%0A)

8. Once you have added steps or rates, you can click on the dot menu to the right of any metric and select the option to **Set sort order**.![Screen_Shot_2023-03-28_at_9.51.46_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733738/5fa0dcf7f1a4b02fa2135457df7b/15263939485459?expires=1778306400&signature=1d4754c59aa1f72682f0df184e4605970b8993caabb1877ad416633c2bbec4bd&req=dSQlF859noZcUfMW3nq%2BgZB3BjIaD1hI4LgAmZTJybJOCJt5kwq6RSuVlngn%0Aq4w%2B%2B0mwv7xD45HGYwQUtQHIpn4%3D%0A)
9. In the **Filters** section at the bottom of the configuration screen you can optionally apply global filters to the data to more precisely define the set of events you are analyzing. For example, if you are looking at a specific program, but want to exclude data from one of the campaigns in that program, you could fill out and add the following filter.![Screen_Shot_2023-03-28_at_9.55.43_AM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733735/ab00b974534a058b3021915c709d/15264068949651?expires=1778306400&signature=e0f1e2fdf3f5072083b0aaf91ac24c0b0e1618b0391b74c61b5e8305a1dc1f35&req=dSQlF859noZcXPMW3nq%2Bgab4EcOayE3PsFJtQGWNM5IxTlcrOR74kMH4WsU%2F%0AcgbTxBwUsi8nWedkSvbydErCpeo%3D%0A)
10. Once you've finished configuring your report, press **Save & Run**.
