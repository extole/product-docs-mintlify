# Overview

The Schedules page, located within the Reports section of your My Extole account, lists all reports that run on a recurring schedule for your programs.

Scheduled Reports in the Extole platform automate and streamline regular report generation. You can set customized parameters and intervals for generating and receiving reports. Stay updated on program performance, events, audiences, and rewards without manual intervention. Save time, improve efficiency, and gain valuable insights regularly for informed decision-making.

## Key Features

### Scheduled Report Details

The reports you see listed are the most recently run versions. If you click on one of them, you will be able to see all of the previously run reports on that schedule as well as general details about the report.

# How to Schedule a Report

To schedule a recurring report you have two options:

## Option 1

1. From the main Reports page, select a category or search for the specific report you want to run.
2. Once you select a report you'll be taken to the configuration screen. Here, choose Schedule instead of Run Once under the Configure Report Options.

## Option 2

1. Navigate to the Scheduled Reports page and hit the + New Scheduled Report button in the top right.
2. You'll be taken to the same report configuration screen, where the Schedule option will already be selected.

![Screen_Shot_2023-05-31_at_4.23.40_PM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421733685/6f5fdf0ea798f0e0dffdc4ff9bd3/17269986411539?expires=1778306400&signature=6a196c275e7801ff9722d5f60b043a2f70b4c3d0a0da9b177cb153151878044b&req=dSQlF859nodXXPMW3nq%2BgerTZ1vkSzYQ%2FGvIsh20YOuv5iGwCdghRLkk%2FA%2BQ%0ATEEB9o2jJvCT%2BO7TNrKIXUxamS4%3D%0A)

Once you've selected the Schedule option several additional fields will appear. You can configure the following elements:

- **Delivery Date**: The date we'll attempt to run and deliver this report. The exact start time will depend on job queue activity.
- **Frequency**: How often the report will automatically generate. Do you want it daily? Weekly? Monthly?
- **Notify Email** (s): The email address(es) that will receive the scheduled report. The people you are sharing the report with do not need My.Extole logins to get the email and access the report; anyone with the publicly-accessible link will be able to see it.
