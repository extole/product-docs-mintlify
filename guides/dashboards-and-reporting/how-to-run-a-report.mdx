# Overview

An essential part of a successful Extole program is your ability to quickly, accurately, and completely report on it. This article describes how to configure and run the various types of Extole reports that give you access to important data generated by your programs.

If you would like a more general introduction to our reports, please reference our [Reports Overview](/guides/getting-help/exploring-the-reports-page).

# Standard Reports

## General Setup

1. On the main Reports page, select your category or use the search bar to find the report you are looking for.![Screen Shot 2023-07-27 at 1.51.32 PM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421734531/a917f121b8395648260b63cefb75/18957552653715?expires=1778306400&signature=798bc9ec751fe442f5d78cd60ca026e336683591f3dfa963cf95fe45ff632392&req=dSQlF859mYRcWPMW3nq%2BgTuWtLL63clFa9aYwHvqvpFCv6xQdO8jj9RL%2FZa4%0AaJCQUvNrcZMu39svjl0bHEYt1R0%3D%0A)
2. Click on the name of the report to open its configuration screen. Here you will see all of the necessary configuration fields you'll need to fill out to run the report.
3. In particular, we recommend editing the Display Name to make the report easier to reference.
4. You can also choose to run the report once, which is our default, or schedule the report so that it runs each week, month, etc.![Screen_Shot_2023-05-31_at_3.05.27_PM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421734533/d18197763cba70847b2537296997/17136040930067?expires=1778306400&signature=ebc0cd70cc2d2651593cd5cdbd71be48785c72c8272bb6f6e93d1473cb3c8e51&req=dSQlF859mYRcWvMW3nq%2BgdH%2BZ%2F8Udy1vgt6LFC9%2BfiNk5qbk0uGhOGivkth8%0AwELC6rEaTQpu7kjdyinWWyjsY8E%3D%0A)
5. In the Filters section, you'll see a variety of fields depending on the type of report. Generally speaking, you'll be able to adjust the time range of data you want to be included in the report and select which programs or campaigns you want to look at.

  - Learn about dates you may be interested in capturing in our article on [Using My Timeline Dates](/guides/dashboards-and-reporting/extole-dashboards-metrics/using-my-timeline-dates).

6. In the Advanced section, you can choose the file format of your report and optionally have it delivered via SFTP if you have a server set up in your account.  
​![Screen_Shot_2023-05-31_at_3.08.45_PM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421734536/50d0c6a2fab73ab2e1edf418873d/17136401471123?expires=1778306400&signature=e84d8f36e1c80952029a0de1892f12d898306700fbe2f5129112c02f0fd06fd9&req=dSQlF859mYRcX%2FMW3nq%2BgQQoXO12RqggPTzbTod9RNKuOen%2F%2F1UzNEkpMphm%0AQR%2FOgd%2FwHqo0HYF4e1ibZcSF0n4%3D%0A)
7. Once you've finished your configuration, hit Save & Run to kick off your report.

## Download Options

Depending on the amount of data included in your report, it may take some time to run. Once it's completed, you can download the report by clicking on it and going to its details page.

If you'd like to create an automated pull on your side, the API call is also available on the details page or within the dot menu on the right-hand side of the reports table.
