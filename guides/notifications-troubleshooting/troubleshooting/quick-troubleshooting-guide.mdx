# Overview

This document is meant to be a quick reference guide to troubleshooting common issues. All steps should be followed sequentially in order to ensure the settings have been properly configured.

# Advocate Share Links

| Problem | Cause | Solution |
| --- | --- | --- |
| Advocate share links are directing friends to a blank page. | Campaign status affects redirection. There should be at least one campaign in either **PAUSED** or **LIVE** status in the program where the share link was generated | In the target program, make sure there is at least one **LIVE** campaign or change the status of an **ENDED** or **STOPPED** campaign to **PAUSED**. |
| The share destination variable in Campaign Edit is empty | Go to the **Friend > Share Destination** creative and add the page where friends should be directed under **Redirect URL.**![Redirect URL or Zone Variable](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421734141/6489d3b100bc43b131181f8987d4/360051581833?expires=1778306400&signature=a92dafe9ad0aa0317c202b21d0f96477615182c29cc6197559ae34ac918bc48e&req=dSQlF859mYBbWPMW3nq%2BgdW%2BgHROYhEHsZAyONkXM8BFGDnfz9Gb6y2LbWz%2F%0ANh71lhC1LQMjR0mBc%2BBm5FT%2B8Ew%3D%0A) |
| The target URL has not been added to **Sites to Enable Javascript** | Login to my.extole and go to the Tech Center[](https://my.extole.com/tech-center) page. If the domain for the **Redirect URL** is not listed in the **Sites to Enable JavaScript,** add it to the list.![Screen_Shot_2019-11-19_at_4.21.12_PM.png](https://extole-5ef307a0e5b1.intercom-attachments-7.com/i/o/syy27wia/1421734144/dd75cf6d62eed5d695b52a9e4103/360050768174?expires=1778306400&signature=fddc748515214d55b15d90b2f89de08ee1b6f5c0e6b1ca279ea90a8256531339&req=dSQlF859mYBbXfMW3nq%2BgaCiURws7IhHS8XZp8erHkyrr6HQRCXaCvaEJE2E%0ANoeC%2FhGzcTb7DhBcM20aRBH2KE0%3D%0A)Only include the second level domain and the top-level domain (see image above). ***Do Not** *include the protocol when whitelisting, i.e.* www, http, https, etc. *|

# Tags / Tagging

| Problem | Cause | Solution |
| --- | --- | --- |
| No CTAs displaying on the client's site | No core.js or CTA is present. | Add both the core.js and the zone tags to your web page. If new ones need to be generated, go to the tag generator[](https://my.extole.com/tech-center/tag-generator) and download them. |
| The zone is disabled in Campaign Edit. | Go to    ​ **Campaign Edit > Marketing > Zone**   and open the creative's kebab menu to make sure it is **Enabled.** Publish the campaign to see the changes reflected on the client's site. |
| The **Serve Creative** toggle is disabled in Campaign Edit. | The **Serve Creative** toggle should be enabled to display the CTA (if hosted by Extole). |
| The campaign contains no zone with that name. | This is usually the case when a campaign is upgraded from scratch. On the new campaign's dashboard, create a new Onsite CTA and publish it. |
| The campaign is missing the default, optional child label and not targeting a campaign. | Reach out to your Extole CSM if you suspect a targeting issue with your campaign. |

# Share Experience

| Problem | Cause | Solution |
| --- | --- | --- |
| Share experience does not send share message | There is a link in the share message to a site not listed under **Production Sites Extole Should Support Requests From** | 1. Make sure to remove the link from the share message. Example, if you are referencing an Amazon gift card as ***Amazon.com*** gift card, you can remove the top-level domain (the .com part).2. If the link in the share message is to a domain you control, make sure to add it to the **Production Sites Extole Should Support Requests From** (Tech Center > Edit Domain). |
| Share Experience/Referral Page does not load | Either core.js tag, zone tag or both tags are missing from the page. | If new ones need to be generated, go to the tag generator[](https://my.extole.com/tech-center/tag-generator) and download. You will need to add these tags to your web pages. |
| The zone is disabled in Campaign Edit. | Go to    ​ **Campaign Edit > Advocate > Zone**   (share experience or referral page, typically) and open the creative's kebab menu to make sure it is **Enabled.** Publish the campaign to see the changes reflected on the client's site. |
